# 🎬 Movie Tracker App

A modern movie tracking app built with Vanilla JavaScript.

## Features

- Search movies with OMDb API
- Select from multiple search results
- Add movies to your collection
- Mark as watched or watchlist
- Add favorites
- Rate movies
- Filter movies
- Store data in localStorage
- Dark mode support

## Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- OMDb API
- LocalStorage
